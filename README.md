### 全国にある花の名所を探すことができる共有サイト「花ナビ(hana_nav)」



# サービス概要

「今住んでいる付近で花を楽しみたい！でも周りにどのようなスポットがあるのか分からない、、」<br>
こんな悩みを解決したい方の手助けになると思い作成しました。

# サービスURL

heroku: https://hana-nav.herokuapp.com/ (レスポンス未対応)

# 環境

- ruby 2.7.2
- rails 6.1
  - Slim(HTMLテンプレートエンジン)
  - bootstrap4(WEBフレームワーク)
  - BEM(CSS設計) 
- MySQL
- heroku(AWS:S3)
